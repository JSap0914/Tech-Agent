# Screen Specifications

## Home Page
- Hero section: 1280x600px, full-width image
- Featured products: 4 cards in a row (desktop), 2 (tablet), 1 (mobile)
- Navigation bar: Sticky, 80px height
- Footer: Links, social media, newsletter signup

## Product Listing Page
- Sidebar filters: 280px width (desktop), collapsible (mobile)
- Product grid: 3-4 products per row
- Pagination: 20 products per page
- Breadcrumbs: Above product grid

## Product Detail Page
- Image gallery: 600x600px main image, 80x80px thumbnails
- Product info panel: Right side (desktop), below images (mobile)
- "Add to Cart" button: Primary color, 48px height
- Reviews section: Below product info

## Shopping Cart
- Cart items: List layout, 100% width
- Each item: 120x120px image, product details, quantity selector
- Cart summary: Fixed on right (desktop), bottom (mobile)
- Empty state: Icon + message + "Continue Shopping" button

## Checkout Page
- Multi-step progress indicator: 4 steps
- Form sections: Shipping, Payment, Review
- Input fields: Full width, 48px height
- Submit button: Full width on mobile, right-aligned on desktop
